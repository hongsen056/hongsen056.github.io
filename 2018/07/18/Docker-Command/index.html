Docker Command

Hongsen
2018-07-18

交互式运行centos

  • docker run -it centos

后台运行容器 可以指定名字

  • docker run -d -name=demo sh_flask

查看全部容器

  • docker container ls -a ps

创建新镜像

  • docker build -t sh/name . 点是当前目录的DockerFile

基于一个镜像创建一个新镜像

  • docker commit b70963b82b27 hongsen056/centos-httpd

删除全部容器

  • docker rm $(docker ps -a -q)

删除全部挂载点

  • docker volume rm $(docker volume ls -q –filter dangling=true)
  • docker system prune -a

查看挂载

  • docker volume inspect …

  • docker volume ls

  • Docker run -d -v $(pwd):容器目录

Docker 网络

查看桥接网络
docker network inspect bridge

  • -e 设置环境变量
  • -d 后台运行
  • -p 本地端口 : 远程端口
  • –link 连接容器
  • docker run -d –link redis –name flask -e * REDIS_HOST=redis -p 5000:5000 hongsen/flask

查看全部容器状态

  • Docker-compose ps
